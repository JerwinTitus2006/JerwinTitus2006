# Security Policy

## Supported Versions

The following versions of this profile and its associated automation workflows are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

I take the security of my repositories and automated workflows seriously. If you find a security vulnerability, please do not open a public issue. Instead, report it directly.

1. **Email**: Send an email to [jerwintitusofficial@gmail.com](mailto:jerwintitusofficial@gmail.com).
2. **Details**: Include a detailed description of the vulnerability, steps to reproduce, and potential impact.
3. **Response**: I will acknowledge receipt of your report within 48 hours and work on a resolution immediately.
